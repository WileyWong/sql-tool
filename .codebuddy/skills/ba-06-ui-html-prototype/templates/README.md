## H5移动端模板

| 模板文件 | 说明 | 适用场景 |
|---------|------|---------|
| [h5/interviewer-h5-prototype.html](h5/interviewer-h5-prototype.html) | 面试官年度回顾H5 | 活动页、营销页、数据报告 |

### 模板特性

- **TDesign CSS变量**：内联完整设计令牌，无需CDN
- **全屏滑动切换**：支持触摸手势（左右滑动翻页）
- **数据卡片组件**：统计数据展示、数字滚动动画
- **页面指示器**：当前页码、导航按钮
- **移动端适配**：max-width: 414px，禁止缩放
- **主题渐变色**：活动主题色变量

### 适用场景

- 年度回顾、数据报告H5
- 活动页、营销页
- 移动端表单、问卷
- 微信H5分享页
