# 安全扫描检查清单

复制此清单跟踪扫描进度。

## 完整扫描流程

```
安全扫描进度:
- [ ] 步骤0: 上下文获取
  - [ ] 读取项目安全配置
  - [ ] 加载白名单规则
  - [ ] 确定技术栈版本
- [ ] ✅ 检查点0: 上下文验证
  - [ ] 知识库读取完成
  - [ ] 扫描规则集已确定
- [ ] 步骤1: 输入分析
  - [ ] 识别输入类型: ___________
  - [ ] 确定目标语言: ___________
  - [ ] 确定扫描范围: ___________
  - [ ] 加载规则模块: ___________
- [ ] ✅ 检查点1: 路由验证
  - [ ] 语言识别正确
  - [ ] 规则模块已加载
  - [ ] 文件列表已确定
- [ ] 步骤2: 执行安全扫描
  - [ ] 入口层扫描完成 (Controller/API)
  - [ ] 业务层扫描完成 (Service/Manager)
  - [ ] 数据层扫描完成 (Repository/DAO)
  - [ ] 工具层扫描完成 (Utils/Helper)
- [ ] 步骤3: 风险评估
  - [ ] 所有问题已分配风险等级
  - [ ] 上下文因子已考虑
  - [ ] 语言特定调整已应用
- [ ] ✅ 检查点2: 报告验证
  - [ ] 扫描完整性确认
  - [ ] 风险统计准确
  - [ ] 修复建议可行
- [ ] 步骤4: 生成报告
  - [ ] 报告结构完整
  - [ ] 修复优先级合理
```
