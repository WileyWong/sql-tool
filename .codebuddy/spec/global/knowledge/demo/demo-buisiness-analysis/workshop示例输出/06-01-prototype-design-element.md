## 第1章：页面结构分析

| 页面名称 | 主要功能 | 目标用户 | 关联旅程阶段 |

## 第4章：设计约束与建议

| 约束类型 | 约束内容 | 建议处理方式 |
|---------|---------|-------------|
| 移动端适配 | H5页面需适配移动端屏幕 | 使用响应式布局，宽度100%，字体使用rem单位 |
| 数据截止日期 | 数据统计截止日期为2025年12月15日 | 页面底部标注数据截止时间 |
| 数据范围 | 仅统计集团正式员工针对集团本部岗位的线上面试数据 | 在页面说明中标注数据范围 |
| 时长格式化规则 | BR5: 年/月无数据时不展示对应单位 | 后端返回格式化字符串，前端直接展示 |
| 年限格式化规则 | BR6: 取年月日展示，年/月为0不展示对应单位 | 后端返回格式化字符串，前端直接展示 |
| 时长单位转换 | BR7: 分钟转小时，四舍五入 | 后端计算后返回 |
| 字数单位转换 | BR8: 字数转万字 | 后端计算后返回 |
| 好评词条类型 | 共4种标准词条类型 | 前端根据词条内容展示对应样式 |
| 性能要求 | 支持10000人并发访问 | 使用缓存策略，静态资源CDN加速 |
| 待确认项 | 分享海报具体设计稿（推断，待确认） | 与业务运营团队确认海报设计 |
| 待确认项 | 低数据版本页面具体UI设计（推断，待确认） | 与UX团队确认低数据版本展示样式 |
