# 前端项目模板

本目录存放前端项目的模板文档。

## 📋 模板列表

- [移动端 H5 模板](./mobile-h5.md) - 移动端 H5 项目模板
- [PC 端模板](./pc.md) - PC 端 Web 项目模板

## 🎯 使用场景

### 移动端 H5 (mobile-h5.md)

适用于:
- 微信公众号 H5 页面
- 移动端活动页
- 移动端营销页
- 响应式移动端应用

### PC 端 (pc.md)

适用于:
- 管理后台
- PC 端官网
- PC 端业务系统
- 桌面端 Web 应用

## 📝 如何使用

1. 根据项目类型选择合适的模板
2. 复制模板内容到项目文档
3. 根据实际需求调整模板内容
4. 补充项目特定的信息
