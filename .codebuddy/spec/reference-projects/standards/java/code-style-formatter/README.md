# 格式化工具

eclipse的格式化工具还在逐步自测，目前版本快速更新，大家使用时候格式化不符合规范的，麻烦提issue或者MR.

## IDEA 插件使用

IDEA的格式化配置文件`Idea-CodeStyle.xml`

- 配置code style
  
 ![code style](https://git.woa.com/standards/java/raw/d7b5e295945b8aa344193f74b2c002d5aabc9954/code-style-formatter/images/idea-code-style-config.png)

- 安装save actions 插件
  
![save actions](https://git.woa.com/standards/java/raw/d7b5e295945b8aa344193f74b2c002d5aabc9954/code-style-formatter/images/save-actions-plugin.png)

- 配置save actions 插件
  
![config save actions](https://git.woa.com/standards/java/raw/d7b5e295945b8aa344193f74b2c002d5aabc9954/code-style-formatter/images/save-actions-config.png)


## eclipse 插件使用


- 配置code style，导入`Eclipse-CodeStyle-xxx.xml`
  
 ![code style](https://git.woa.com/standards/java/raw/4739bd0d472312cc7ce57eafff80fdf1b7d488d6/code-style-formatter/images/eclipse-code-style-config.png)

- Import 格式化规则导入`Eclipse-CodeStyle-OrganizeImport`
  
![import规则](https://git.woa.com/standards/java/raw/4739bd0d472312cc7ce57eafff80fdf1b7d488d6/code-style-formatter/images/eclipse-import-config.png)

- 配置save actions 插件
  
![config save actions](https://git.woa.com/standards/java/raw/4739bd0d472312cc7ce57eafff80fdf1b7d488d6/code-style-formatter/images/eclipse-save-actions-config.png)
