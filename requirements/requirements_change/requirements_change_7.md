## 1. 变更内容

### 1.1 功能需求

> 详细描述需要实现的功能

| 序号 | 功能点 | 描述 | 验收标准 |
|------|--------|------|----------|
| 1 | SQL编辑器SQL语法样式 | 字符串现在是大红色，看起来太显眼了，改成橙色，颜色值：#f5b801 | |
| 2 | 新增、修改且字段时无法输入DATETIME(fsp)类型 | DATETIME可以指定精度，现在无法实现 | |
| 3 | 结果表格(ResultTable.vue)中日期格式 | 根据数值类型，格式化，<br/>DATE显示成yyyy-MM-dd，<br/>DATETIME显示成：yyyy-MM-dd HH:mm:ss，<br/>DATETIME(fsp)根据fsp的值，在DATETIME基础上显示到毫秒值,<br/>TIME显示成HH:mm:ss，YEAR显示成yyyy | |
| 4 | 结果表格(ResultTable.vue)中增加“查看”弹出层 | 右键点击结果表中的单元格，显示“查看”菜单，点击后，弹出层，显示其中的内容，上方有选项：原始值（显示成原始值）、json（转变成json格式化显示）、xml(转变成xml格式化显示)。按ESC可以直接退出，或者点击“关闭”按钮也可退出 | |
