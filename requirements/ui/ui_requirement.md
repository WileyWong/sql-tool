## 概述

实现一个sql编辑器的UI

这个文档描述UI界面如何实现

目的：按这个文档生成一个HTML原型，并输出到 requirements/ui/output文件夹

## 框架大概长这样

|窗口标题: sql-tool  |                                                          |
| --------------------|------------------------------------------------------- |
|菜单栏               |                                                         |
|功能按钮栏            |                                                        |
|数据库连接树形展示   | sql1.txt\sql2.txt\...                                  |
|                    |当前连接的数据库服务器 当前连接的数据库名字 当前使用的用户名 最大行数|
|                    |执行结果                                                 |
|最底部的状态栏|

## 细节调整
1. 数据库连接树形展示区域可左右调整大小。 执行结果可上下调整大小。
2. 数据库连接树形展示结构：服务器地址->数据库名->Schema（mysql没有，sqlserver有，做两个演示出来）->表、视图、存储过程、函数
   1. 数据库连接树形展示 增加展开、折叠功能。并在菜单的“查看”里增加展开、折叠全部选项
3. 执行结果 中的表格可以左右拖动列大小
4. 当前连接的数据库服务器 当前连接的数据库名字 可下拉更换其他值
5. 增加新建连接的弹框，要求有：名称、服务地址、用户名、密码、默认数据库名（可选）。 功能有：测试连接，保存、取消。必填项后面加上红色的*标记
6. 数据库连接树形展示 鼠标划过服务地址时，后面出现修改名称、删除 图标，点修改名称后，服务器地址这里可编辑，输入回车后完成编辑。点删除时，二次提示用户后删除。
   1. 点删除时，二次提示用户后删除 使用HTML弹窗（参考element组件)，不要用浏览器的。
7. sql编辑区显示行号
8. 鼠标在编辑区移动时，在最底部的状态栏更新当前所在行、列。
9. sql编辑区域，在服务器、数据库、用户后面增加"最大行数"选项：
   - 默认值：5000
   - 只允许输入数字
   - 用户可自行修改
   - 用于限制查询结果返回的最大行数
10. 数据库连接树形展示 树节点右键菜单增加"刷新"选项：
    - 右键任意节点显示上下文菜单
    - 点击"刷新"后重新读取数据库结构
    - 刷新指定节点下的内容

再生成特殊情况的原型文件 index_empty.html
1. 没有保存过任何数据库连接时
   1. 数据库连接树形展示 区分展示提示“未连接数据库，点击新建连接”。 点“新建连接”后，弹出新建连接弹框
   2. 右侧展示提示“无数据库连接”即可
